# Demo 页面模块拆解

## 1. CMS 数据初始化页

截图：

![CMS 初始化空状态](assets/22-cms-init-empty.png)

![CMS 产业链推荐](assets/23-cms-chain-recommendations.png)

![CMS 已选择产业链](assets/24-cms-chain-selected.png)

### 页面区域

| 区域 | Demo 表现 | 生产化要求 |
| --- | --- | --- |
| CMS 左侧导航 | 静态展示 CMS 管理菜单 | 后续接入真实菜单权限，本期可保持固定入口 |
| 顶部面包屑 | 平台管理 / AI 课程管理 | 后续按真实后台信息调整 |
| 专业建设 Tab | 展示多个专业建设 Tab | 本期只要求产业调研管理可用 |
| 初始化说明 | 说明数据初始化用途 | 文案保留，补专业名称和状态 |
| 数据初始化按钮 | idle 状态显示“数据初始化” | 调用初始化接口 |
| 重新初始化按钮 | ready 状态显示“重新初始化” | 需要确认弹窗或批次记录 |
| 初始化中状态 | 显示进度条和说明 | 后端任务轮询或异步状态刷新 |
| 推荐产业链列表 | 展示匹配度、摘要、理由、标签 | 后端返回推荐列表 |
| 选择按钮 | 可选择默认产业链 | 保存专业默认产业链 |

### 交互状态

| 状态 | 触发 | 页面反馈 |
| --- | --- | --- |
| idle | 专业未初始化 | 展示待初始化说明和主按钮 |
| initializing | 点击数据初始化 | 展示进度条、隐藏推荐列表 |
| ready | 初始化完成 | 展示推荐产业链列表 |
| selected | 点击选择产业链 | 行高亮，前台可读取默认产业链 |
| failed | 初始化失败 | 生产化需补失败提示和重试 |

## 2. 岗位画像分析页

截图：

![岗位画像分析](assets/05-job-portrait.png)

### 页面区域

| 区域 | Demo 表现 | 生产化要求 |
| --- | --- | --- |
| 页面标题 | 岗位画像分析 | 保留 |
| 当前产业链 | 智能建造产业链 | 从 CMS 默认产业链读取 |
| AI 洞察 | 静态洞察列表 | 后续可由规则或 AI 生成 |
| KPI 卡片 | 岗位、任务、能力项、证书 | 后端汇总或前端按详情汇总 |
| 搜索框 | 搜索岗位、技能、产业链环节 | 接口查询或前端过滤 |
| 岗位等级筛选 | 全部、初级、中级、高级 | 接口参数或前端过滤 |
| 岗位卡片 | 薪资、需求量、等级、标签 | 来自岗位画像列表 |
| 分页 | 当前 demo 有分页逻辑 | 后端分页或前端分页 |

### 岗位卡片字段

| 字段 | Demo 来源 | 说明 |
| --- | --- | --- |
| 岗位名称 | `PORTRAIT_JOB_PROFILES.name` | 卡片主标题 |
| 薪资区间 | `salary` | 例如 10K-18K |
| 需求量 | `demand` | 与招聘趋势同口径更好 |
| 岗位等级 | `level` | 初级、中级、高级 |
| 产业环节 | `chain` | 实际更接近产业链/环节说明 |
| 技能标签 | `skills` | 取岗位能力前几项 |

## 3. 岗位画像详情弹窗

截图：

![岗位画像详情](assets/06-job-portrait-detail.png)

### 页面区域

| 区域 | Demo 表现 | 生产化要求 |
| --- | --- | --- |
| 详情头部 | 岗位名称、来源说明、关闭按钮 | 保留 |
| 基本信息 | 薪资、学历、经验、等级、需求量 | 后端岗位详情字段 |
| 岗位摘要 | 文本说明岗位职责 | 后端维护或 AI 生成后审核 |
| 能力要求 | 知识、技能、素养分组 | 标准能力项库 |
| 能力雷达 | 5 个维度雷达图 | 可先按配置分值展示 |
| 典型任务 | 任务标签或清单 | 后续进入岗位建设中心复用 |
| 证书 | 证书名称、等级 | 后续可展开证书详情 |
| 关联专业 | 专业或方向标签 | 后续接专业库 |
| 企业样本 | 企业名称、行业、地区、标签 | 后续接企业样本库 |

### 详情弹窗一期边界

- 做查看，不做编辑。
- 做字段完整展示，不做复杂审批。
- 保留证书详情、企业详情后续扩展入口，但不作为 P0。

## 5. 页面之间的联动

| 联动 | Demo 状态 | 一期生产化要求 |
| --- | --- | --- |
| CMS 默认产业链 -> 岗位画像 | Demo 为前端静态状态 | 后端保存并由前台读取 |
| CMS 默认产业链 -> 招聘趋势 | Demo 为同一演示上下文 | 招聘接口必须接收 `chainId` |
| 招聘明细岗位 -> 岗位画像 | Demo 暂未强跳转 | P1 可做点击跳转画像详情 |
| 岗位画像能力 -> 技能热度 | Demo 命名相近 | 生产化需要标准能力项映射 |
